Kcal - Smart Parking System

Description

An IoT-based smart parking system that helps users find available parking spots in real-time. The system uses sensors to detect vehicle presence and provides mobile app integration for seamless parking experience.

Technologies Used

  • Flutter
  • Firebase
  • Dart
  • Arduino
  • IoT Sensors
  • Google Maps API

Key Features

  • Real-time parking spot detection
  • Mobile app for spot reservation
  • Digital payment integration
  • Parking history and analytics
  • Admin dashboard for management
